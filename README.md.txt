# Smart Door Lock System

An IoT-based smart door locking system using ESP32 and RFID authentication.

---

## 🚀 Features

- RFID card authentication
- Automatic door locking/unlocking
- Servo motor control
- Unauthorized access alert using buzzer
- LCD status display
- ESP32 based system

---

## 🛠 Components Used

- ESP32
- RFID RC522
- Servo Motor
- LCD Display
- Buzzer
- Jumper Wires

---

## ⚙️ Working

1. User scans RFID card
2. ESP32 verifies card ID
3. If authorized:
   - Servo motor unlocks door
4. If unauthorized:
   - Buzzer alert activates
